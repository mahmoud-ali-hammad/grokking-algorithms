# Grokking Algorithms — Notes, Explanations & Implementations (C++)

This repository contains my study notes, explanations, and **C++** implementations based on the book **_Grokking Algorithms_** (Aditya Y. Bhargava).

The goal is to:

- Explain each idea in simple words (the “grokking” style)
- Add clear examples and illustrations (when helpful)
- Provide working implementations and practice problems in **C++**
- Keep everything organized by chapter/topic
