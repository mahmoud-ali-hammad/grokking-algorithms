# Grokking Algorithms (C++)

This repository contains my C++ implementations and practice exercises based on the book **Grokking Algorithms**.

## 📚 Repository Structure

```text
grokking-algorithms/
├── chapter-1/
├── chapter-2/
├── chapter-3/
├── chapter-4/
└── Readme.md
```

Each chapter folder includes code examples and algorithm implementations related to that chapter’s concepts.

## 🧠 Covered Topics

- **Chapter 1**: Introduction to Algorithms / Binary Search
- **Chapter 2**: Selection Sort
- **Chapter 3**: Recursion
- **Chapter 4**: Quicksort

> Topics may expand as more chapters are added.

## ⚙️ Language

- **C++**

## 🚀 How to Run

Compile and run any C++ file using `g++`:

```bash
g++ file.cpp -o output
./output
```

Example:

```bash
g++ chapter-1/binary_search.cpp -o binary_search
./binary_search
```
